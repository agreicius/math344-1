Implication: (ii) \(\implies\) (i).
We prove the contrapositive. Suppose \(X\) is not compact, and let \(\mathcal{U}=\{U_i\}_{i\in I}\) be a cover with no finite cover. Define \(J\) to the be the set of all pairs \((K, x_K)\text{,}\) where \(K\) is a finite subset of \(I\) and \(x_K\in X-\bigcup_{k\in K}U_k\text{.}\) (Note that \(X-\bigcup_{k\in K}U_k\ne\emptyset\) since \(\mathcal{U}\) has no finite subcover.) It is easy to see that \(J\) is a directed set under the quasiordering defined as \((K, x_K)\leq (K', x_{K'})\) if and only if \(K\subseteq K'\text{.}\) Define the net \(f\colon J\rightarrow X\) as \(f((K, x_K))=x_K\text{.}\)
