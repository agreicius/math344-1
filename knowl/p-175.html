When \(X=\emptyset\text{,}\) we have \(\mathcal{P}(X)=\{\emptyset\}\text{.}\) Since by axiom (1) a topology \(\mathcal{T}\) on \(X\) must contain both \(\emptyset\) and \(X=\emptyset\text{,}\) we see the only option is \(\mathcal{T}=\{\emptyset\}\text{.}\) It is now straightforward to show that \(\mathcal{T}=\{\emptyset\}\) is a topology on \(X=\emptyset\text{.}\) Indeed, it satisfies axiom (1), since it includes both \(\emptyset\) and \(X=\emptyset\) (which happen to be the same thing in this example) satisfies axioms (1)-(3). Indeed, we have \(\emptyset\in \mathcal{T}\) and \(X\in \mathcal{T}\text{,}\) since \(X=\emptyset\) in this case; and since \(\mathcal{T}\) only contains one element, it is clearly closed under unions and finite intersections.
