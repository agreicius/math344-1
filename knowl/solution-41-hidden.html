Let \(\mathcal{U}=\{U_i\}_{i\in I}\) be an open covering of \(Y\text{,}\) so that \(Y\subseteq \bigcup_{i\in I}U_i\text{.}\) We have \(0\in U_{i_0}\) for some \(i_1\in I\text{.}\) Since \(U_{i_0}\) is open, there is an \(N\) such that \((-1/N, 1/N)\subseteq U_{i_0}\text{.}\) It follows that \(U_{i_0}\) contains all of \(Y\) except possibly the elements \(\{1, 1/2, \dots, 1/N\}\) Now pick open sets \(U_{i_k}\) containing \(1/k\) for each \(1\leq k\leq n\text{.}\) It follows that \(Y\subseteq \bigcup_{k=0}^NU_{i_k}\) is a finite subcovering of \(\mathcal{U}\text{.}\)
